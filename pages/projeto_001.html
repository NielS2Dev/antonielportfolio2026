Projeto: Régua Digital com Sensor Ultrassônico e Arduino

A ideia aqui foi criar um medidor de distância em tempo real usando o ARDUINO UNO R3. É um projeto prático que mostra como transformar sinais sonoros em dados que no caso do meu exemplo será a distância do objeto.

O Hardware que eu usei foram os seguintes equipamentos.

  • Arduino UNO 📟 O Cérebro do projeto
  • Display 16x2 📱 O "Monitor" para ver as informações em tempo real
  • Protoboard 🧱 Placa para fazer as conexões dos dispositivos e equipamentos
  • Fios/Jumps 🧵
  • Sensor De Distância Ultrassônico - HC-SR04 📡 Seria o "OLHO" que mede a distância do objeto para o sistema identificar
  • Álcool isopropílico 🧪🔌 Foi essencial para limpar meus componentes, pois boa parte estava oxidado.

Explicação do Projeto

O Princípio do Ultrassom

O Sensor HC-SR04 Funciona de forma similar a um sonar de um morcego ou de um submarino. Ele possui dois "olhos" se é que podemos chamar assim em que um seria o emissor e o outro o receptor.

  • Emissão

    O Emissor envia um pulso sonoro de alta frequência (ultrassom)

  • Eco

    Esse som viaja pelo ar, bate em um objeto e retorna para o receptor

A Lógica do Código

O Arduino atua como um cronômetro de precisão, em que ele mede exatamente quanto tempo o som levou para ir e voltar, observação que durante a programação podemos acabar usando as unidades de medidas milímetros, centímetros e metros, e acabou sendo melhor CM, já que por ser um dispositivo barato, ele não é tão preciso para milímetros e muito menos para metros, logo em meus testes o que melhor funcionou foi usando a como medida a unidade de centímetros. Outro detalhe seria que como o a velocidade do som no ar é de aproximadamente 340m/s, o código faz uma continha...

  • Multiplica o tempo pela velocidade do som
  • Dividimos por 2(Já que o som faz o caminho de ida e volta)

A Exibição em Tempo Real

Após o cálculo, o valor é convertido para centímetros e enviado para o display LCD 16x2. O Código limpa a tela e atualiza o valor a cada 200 milissegundos, permitindo uma leitura de certa forma fluida e precisa, na condição de centimetros.

Código Fonte do Projeto(Arduino IDE) 💻


    #include  <HCSR04.h>
    #include  <LiquidCrystal_I2C.h>

    const byte pinTrig = 8;
    const byte pinEcho = 9;

    LiquidCrystal_I2C lcd(0x27,2,1,0,4,5,6,7,3, POSITIVE);

    double *result;

    void setup(){
    HCSR09.begin(pinTrig, pinEcho);

    lcd.begin(16,2);
    lcd.backligh();
    }

    void loop(){
    result = HCSR09.measureDistanceCm();

    lcd.setCursor(0, 0);
    lcd.print("Distância:");
    lcd.print(result[0], 0);
    lcd.print(" cm    ");

    lcd.setCursor(1, 1);
    lcd.print("Régua do Niel");

    delay(200)
    }
